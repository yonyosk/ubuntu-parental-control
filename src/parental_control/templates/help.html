Help & Documentation

Getting Started

Ubuntu Parental Control helps you manage and monitor internet access on your system. Here's how to use the main features:

Dashboard

The dashboard shows your system's current status, including protection status, blocked domains count, and today's usage statistics.

DNS Settings

Configure DNS servers for content filtering. You can use:

  • System DNS: Use your system's default DNS settings
  • OpenDNS: Family-friendly filtering (208.67.222.222, 208.67.220.220)
  • Cloudflare for Families: Malware and adult content blocking (1.1.1.3, 1.0.0.3)
  • Custom: Specify your own DNS servers
Content Filtering

Block specific websites or entire categories:

  • Add individual domains to block specific sites
  • Enable category-based blocking (social media, gambling, adult content, etc.)
  • View and manage your blocked sites list
Time Management

Set time-based restrictions and usage limits to control when and how long internet access is allowed.

Reports

Monitor internet activity with detailed reports showing blocked attempts, usage patterns, and access statistics.

Quick Actions
Block a Website

Go to Content Filtering and add the domain name (e.g., example.com) to your blocked sites list.

Enable Safe DNS

Go to DNS Settings and select "Cloudflare for Families" for automatic filtering of malicious and adult content.

System Requirements
  • Ubuntu 18.04+
  • Root privileges
  • Python 3.6+
  • Network access
Important Notes
  • • Changes may require system restart
  • • DNS changes affect all users
  • • Backup your settings regularly
  • • Monitor logs for issues
Troubleshooting

If you experience issues:

  • 1. Check system logs
  • 2. Verify DNS settings
  • 3. Restart the service
  • 4. Clear browser cache
Frequently Asked Questions
